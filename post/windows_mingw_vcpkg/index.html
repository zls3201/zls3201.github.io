Windows 10 mingw vcpkg配置

  • zls3201
  • dev
  • 2021-02-24

安装 Git

安装 CMake

安装 vcpkg

查看triplet

vcpkg help triplet

配置环境变量 VCPKG_DEFAULT_TRIPLET

VCPKG_DEFAULT_TRIPLET = x64-mingw-dynamic

安装配置mingw-distro

解压Mingw
添加”bin” “x86_64-w64-mingw32\bin”两个子目录到Path

测试

vcpkg install sdl2:x64-mingw-dynamic
